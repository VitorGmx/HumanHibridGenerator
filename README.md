Um gerador interativo de criaturas híbridas humanoides, projetado para criação rápida de personagens, conceitos visuais e worldbuilding.

O sistema combina atributos físicos, estéticos e mutações aleatórias para gerar entidades únicas de forma controlada.

---

## ⚙️ Funcionalidades

- Geração de criatura baseada em categorias independentes
- Bloqueio de escolhas após seleção (sistema de decisão única)
- 4 atributos extras aleatórios sem repetição
- Animação simples de seleção (efeito de roleta)
- Interface leve e responsiva
- Execução 100% client-side (sem backend)

---

## 🧬 Sistema de Geração

Cada criatura é composta por camadas de atributos:

### 🔹 Estrutura base
Define a espécie principal:
- Humano
- Elfo
- Orc
- Goblin
- Vampiro
- Anjo
- Demônio
- Gigante
- Sereia
- Lobisomem

---

### 🔹 Atributos físicos
- Gênero
- Rosto (estrutura facial)
- Olhos (condição e tipo visual)
- Nariz (formato)
- Cabelo (cor, estilo e textura)

---

### 🔹 Estética
- Vestimenta (estilo visual simplificado)
- Pose (posição corporal)

---

### 🔹 Extras (mutação)
Cada criatura recebe 4 atributos únicos adicionais, como:
- chifres
- asas
- ossos expostos
- fumaça
- cristais
- musgo
- energia instável

---

## 📜 Regras do sistema

- Cada categoria só pode ser selecionada **uma vez**
- Seleções são permanentes após confirmação
- Extras não podem se repetir dentro da mesma criatura
- O sistema prioriza consistência visual básica entre atributos
- Todas as gerações são aleatórias dentro de listas predefinidas
